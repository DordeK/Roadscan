Architecture

Crowdsourcing that
gets smarter over time.

📱 Mobile App
  • React Native + Expo — iOS & Android
  • Accelerometer at 20 Hz, background GPS
  • Surface classification every 2s
  • Audio warnings within 100m
⚙️ Backend
  • Node.js + PostGIS spatial clustering
  • Deduplicates reports within 15m radius
  • ONNX inference in <100ms
  • Daily confidence decay — stale data auto-expires
🛣️ Road Segmentation
  • Parallel to detection — runs every 2 seconds
  • Decision tree classifier on 40-sample window
  • 4 surface types: smooth, rough, cobblestone, gravel
  • Builds a continuous segment map alongside potholes
🗄️ Data Model
  • PostgreSQL with GEOMETRY(POINT) indexes
  • Confidence grows with each new confirmation
  • Severity upgrades after 2+ devices agree
  • Device UUIDs only — no PII collected
Confidence formula
confidence = min(99, 30 × ln(reports + 1))
1 report → 21% · 5 reports → 49%
Never reaches 100% — always open to revision
